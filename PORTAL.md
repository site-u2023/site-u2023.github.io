# script.js 解説

### アドレス
- `IPアドレス`： (192.168.1.1 等)
- `+` / `-`：アドレス追加・削除
- `適用`：設定反映

### ブラウザ
- `サービス`：（LuCI / ttyd / filebrowser / AdGuardHome / netdata）
- `+` / `-`：サービス追加・削除
- `ポート`：ポート番号
- `適用`：設定反映
- `開く`：ブラウザ起動
- `QRコード`：QRコード表示

### ターミナル (Windows用)
※要IPアドレス選択（または追加）

- `タイプ`（aios2 / aios / ssh）
- `ダウンロード`：`.bat`ファイル取得

### キッティングツール（カスタムビルダー）
`デバイス用のOpenWtファームウェアをダウンロード`：カスタムビルドツールリンク
