# aios2 インストーラー

## このインストーラーでできること

Windows PC から OpenWrt ルーターへ SSH 接続し、自動設定を行うツールをインストールします。

### インストールされる機能

インストール時に3つの機能から選択可能：

| 機能 | 説明 |
|------|------|
| **aios2** | OpenWrt 自動設定スクリプト実行 |
| **aios** | 旧版スクリプト実行 |
| **SSH** | OpenWrt ルーターへ直接 SSH 接続 |

各機能は**スタートメニュー**と**デスクトップ**にショートカット作成されます。

## インストール方法

1. `aios2.msi` をダブルクリック
2. インストールウィザードに従う
3. 必要な機能を選択
4. インストール完了

## 使い方

### aios2 を使う場合

1. デスクトップまたはスタートメニューから「**aios2**」を起動
2. OpenWrt ルーターの IP アドレスを入力（自動検出されます）
3. root パスワードを入力（初回のみ）
4. 自動的に設定スクリプトが実行されます

### SSH で直接接続する場合

1. デスクトップまたはスタートメニューから「**SSH**」を起動
2. OpenWrt ルーターの IP アドレスを入力
3. root パスワードを入力
4. コマンドラインで操作できます

## システム要件

- Windows 10/11（64bit）
- OpenSSH クライアント（Windows 10 1809 以降は標準搭載）
- OpenWrt ルーターへのネットワーク接続

## インストール先

```
C:\Program Files\AIOS2\
  └─ aios2.exe
```

## アンインストール

Windows の「設定」→「アプリ」→「aios2」から削除できます。
